
# BTCheck — Bitcoin News & Data Platform

## Overview

BTCheck is a personal project designed to simplify access to **Bitcoin news and market information**.

The platform aggregates news, provides real-time pricing data, and allows users to convert FIAT currencies to BTC.

## Architecture

The system was designed with automation and efficiency in mind.

Main components include:

- Python backend
- Automated web scraper
- Dedicated API
- Lightweight frontend data consumption

## Key Features

- Daily Bitcoin news aggregation
- Real-time price tracking
- FIAT to BTC converter
- Historical news query via API

## Quality Approach

The project includes **automated testing and validation pipelines**.

Tests are implemented using:

- Robot Framework
- Cypress

These tests validate both API responses and frontend behavior.

## Technologies

- Python
- Web scraping pipelines
- JSON data generation
- Robot Framework
- Cypress
