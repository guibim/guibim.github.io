# CONV+ — Modular File Conversion API

## Overview

CONV+ is a modular web platform designed for **file format conversion through a scalable API architecture**.

The project started with a need to convert **Stata (DTA) datasets into CSV**, later evolving into a more extensible conversion platform.

## Architecture

The backend is built with **FastAPI** and follows a modular processing architecture where each converter operates as an independent processing unit.

Key design goals:

- Low coupling
- Modular expansion
- Lightweight runtime environment
