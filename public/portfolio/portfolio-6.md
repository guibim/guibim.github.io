# Per5HUB — OpenBIM Project Management Platform

## Overview

Per5HUB is a custom-built project management platform developed to support **OpenBIM project workflows and collaboration**.

The platform centralizes validation processes, project management tools, and team coordination features.

## My Role

Participated in the design and development of the platform structure and workflow logic.

Key responsibilities included:

- Defining system requirements based on OpenBIM workflows
- Designing process traceability for BIM validation tasks
- Structuring collaboration tools for engineering teams

## Platform Features

- IFC validation workflows
- IDS compliance checking
- BCF issue management
- OpenCDE integration concepts
- Project management tools (Kanban & Gantt)

## Quality Approach

The system was designed with a **quality-first mindset**, focusing on traceability, documentation, and collaborative problem resolution.

Key principles include:

- Clear issue tracking
- Standardized validation workflows
- Transparent project progress tracking

## Technologies

- Web-based project platform
- BIM workflow integrations
- Collaborative project tools
