# SCOPE — Project Management for Service Offices

> End-to-end system for project-based businesses, from first contact to final payment.

## Overview

SCOPE is a web application built for architecture firms, engineering offices, consultancies, and any service business that works in project cycles.

It covers the full commercial and operational cycle: lead capture, proposal generation, project execution tracking, financial control, and deadline management — with everything connected in a single workflow instead of scattered across CRM, spreadsheet, and email.

## Key Features

- **Lead CRM** — capture and qualify potential clients, track pipeline stages
- **Versioned Proposals** — create, version, and export professional proposals as PDF
- **Per-project Kanban** — task board scoped to each project with assignees and deadlines
- **Financial Tracking** — link invoices, payments, and milestones to each project
- **Deadline Management** — project timeline with alerts for upcoming and overdue deliverables
- **Team Collaboration** — multi-user workspace with role permissions

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + TypeScript |
| Backend / Auth | Supabase (PostgreSQL + Row Level Security) |
| PDF Generation | React-PDF / browser print |
| Styling | Tailwind CSS |
